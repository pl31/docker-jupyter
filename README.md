docker-chrome
=============

This is a debian image and jupyter notebook installed.

Images and Tags
---------------

Following images are build on [docker hub](https://hub.docker.com/r/pl31/jupyter/tags/).

- `pl31/jupyter:jessie`

Use image
---------

```
docker run -p 127.0.0.1:8888:8888 pl31/jupyter:jessie`
```
